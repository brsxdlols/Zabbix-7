\# 🚀 Zabbix 7 LTS - Instalador Automático para Debian 12



Instalador totalmente automatizado do \*\*Zabbix Server 7.0 LTS\*\* para \*\*Debian 12 (Bookworm)\*\*.



O objetivo deste projeto é realizar uma instalação completa do Zabbix com um único comando, configurando automaticamente todos os componentes necessários.



\---



\## Recursos



\- ✅ Zabbix Server 7.0 LTS

\- ✅ PostgreSQL

\- ✅ NGINX

\- ✅ PHP-FPM

\- ✅ Zabbix Frontend

\- ✅ Zabbix Agent 2

\- ✅ SNMP

\- ✅ SNMP Trap

\- ✅ Fping

\- ✅ Traceroute

\- ✅ Configuração automática do banco de dados

\- ✅ Ajuste automático do PostgreSQL

\- ✅ Configuração automática do PHP

\- ✅ Configuração automática do NGINX

\- ✅ Frontend configurado automaticamente

\- ✅ Serviços iniciados automaticamente



\---



\## Sistemas suportados



\- Debian 12 (Bookworm)



\---



\## Instalação



Execute apenas um comando:



\### Utilizando CURL



```bash

bash <(curl -fsSL https://raw.githubusercontent.com/brsxdlols/Zabbix-7/main/instalar-zabbix-7-debian12.sh)

```



\### Utilizando WGET



```bash

bash <(wget -qO- https://raw.githubusercontent.com/brsxdlols/Zabbix-7/main/instalar-zabbix-7-debian12.sh)

```

Admin

zabbix
